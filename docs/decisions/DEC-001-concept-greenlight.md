# DEC-001: Concept greenlight

- Status: Accepted
- Date: 2026-07-21
- Decision owner: User/project owner

## Decision

Approve **Logan and the Aetherlings** for preproduction: angled/top-down stylized 3D, strategic turn-based battles, connected scenes, family-friendly tone, Windows/iPhone PWA, local saves before optional cloud sync, public GitHub repository, and an eventual 60–80-creature target.

Capture uses triangular **Aether Prisms**. Players weaken a wild Aetherling, may improve odds with status/equipment, throw the Prism, and resolve success or failure. Friendship may exist separately but is not required to capture.

## Consequences

Preproduction may build and test the minimum loop. Full production awaits mobile performance, fun, save reliability, and one MeshyAI production-pipeline proof.
